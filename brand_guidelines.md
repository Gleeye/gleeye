# Brand Guidelines: Gleeye ERP

**Goal**: Maintain a premium, modern, and cohesive aesthetic. "Ragionato e meno banale".

## 1. Typography
- **Primary Font**: `Outfit` (sans-serif)
    - Used for both Titles and Body.
    - Weights: Light usage of bold. Prefer font-weight 400/500.

## 2. Color Palette
### Brand Identity
- **Brand Blue**: `#4e92d8` (rgb: 78, 146, 216)
- **Brand Viola**: `#614aa2` (rgb: 97, 74, 162)
- **Gradient**: `linear-gradient(135deg, #4e92d8 0%, #614aa2 100%)` (Signature look)

### Theme: Light (Default)
- **Background**: `#fafbfc` (Clean, off-white)
- **Card Background**: `#ffffff`
- **Text**:
    - Primary: `#1a1f36`
    - Secondary: `#697386` (Subtitles, labels)
    - Tertiary: `#a0aec0`
- **Borders**: Glass-like (`rgba(0, 0, 0, 0.06)`)

### Theme: Dark
- **Background**: `#0a0a0a`
- **Card Background**: `rgba(25, 25, 25, 0.6)`
- **Text**: `#f8fafc` (Primary)

## 3. Design Tokens
- **Radius**:
    - XL: `32px` (Modals, big cards)
    - LG: `24px`
    - MD: `16px` (Standard cards)
    - SM: `8px` (Buttons, inputs)
- **Shadows**:
    - Premium: `0 2px 8px rgba(0, 0, 0, 0.06)`
    - Soft: `0 1px 3px rgba(0, 0, 0, 0.04)`

## 4. UI Rules
- **Micro-interactions**: Hover effects, smooth transitions (`0.3s`).
- **Glassmorphism**: Use `backdrop-filter: blur()` effects for sidebars and overlays.
- **Icons**: Minimalist, stroke-based.
- **Buttons**:
    - Primary: Gradient background, white text, premium shadow.
    - Secondary: Transparent/Glass bg, text color.
